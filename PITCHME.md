# Dashboard Pimpinan
Aplikasi Monitoring untuk Pimpinan
---
## Dashboard Pimpinan
- Dashboard Pimpinan merupakan aplikasi antar muka untuk monitoring dan mempermudah Pimpinan untuk melihat dan menilai kinerja OPD (Organisasi Perangkat Daerah). |
- Pada aplikasi ini Pimpinan dapat melihat kinerja OPD dalam bentuk prosentase, grafik dan list data. |

---

## Cara Penggunaan Aplikasi Dashboard Pimpinan

+++
- Bukalah aplikasi Dashboard Pimpinan melalui web browser dengan alamat url sebagai berikut : http://dashboard-01.dev.bantenprov.go.id/#/login . |
- Kemudian tekan Enter pada tombol keyboard atau klik tombol GO pada browser. |
- Maka pada layar akan tampak menu login situs Dashboard Pimpinan seperti ini: |

+++
![tampilan login](/assets/images/dashboard-login.png) 

+++
Masukan Email Id dan Password, setelah dimasuka dengan benar, klik tombol Log in atau tekan tombol Ente pada keyboard. Sehingga akan menampilkan halaman utama.

#### 2.2.3 Data Kepegawaian
Pada menu Data Kepegawaian ini akan memperlihatkan data master dari sub menu Kepegawaian, yang antara lain:
1. List Pegawai
2. List Pegawai KPO
3. List Pegawai Esselon
4. List Pangkat Pegawai
5. List Usia ASN
6. List Pegawai yang akan Pensiun

##### 2.2.3.1 List Pegawai
List Pegawai dapat dilihat dengan cara mengklik sub menu "List Pegawai" seperti pada gambar dibawah ini:
[![Klik List Pegawai](../images/klik-list-pegawai.png)](../images/klik-list-pegawai.png)

Maka aplikasi akan menampilkan tabel list pegawai seperti gambar dibawah ini:
[![Tabel List Pegawai](../images/data-kepegawaian-list-pegawai.png)](../images/data-kepegawaian-list-pegawai.png)
Gambar 21. Tabel List Pegawai

Didalam tabel ini *user* dapat melihat data jumlah pegawai, pada halaman ini *user* juga dapat menambahkan, mengedit dan menghapus data jumlah pegawai dari aplikasi Dashboard Pimpinan ini.

###### 2.2.3.1.1 Tambah Data Jumlah Pegawai
*User* dapat menambahkan data di sub menu List Pegawai ini dengan cara mengklik tombol "Create" seperti gambar dibawah ini:

[![Tombol Klik Tambah Data](../images/tombol-tambah-data.png)](../images/tombol-tambah-data.png)

Sehingga akan menampilkan *Form Tambah Jumlah Pegawai* seperti gambar dibawah ini:

[![Form Tambah Pegawai](../images/form-kepegawaian-tambah-pegawai.png)](../images/form-kepegawaian-tambah-pegawai.png)
Gambar 22. Tambah Jumlah Pegawai

Pada halaman ini *user* dapat melihat *form* untuk menginput data jumlah pegawai, setelah mengisi *form, user* dapat mengklik tombol "Submit" untuk menyimpan data, tombol "Reset" untuk menghapus data yang sudah diinput atau tombol "Back" untuk kembali ke halaman List Pegawai tanpa menyimpan data.

###### 2.2.3.1.2 Edit Data Jumlah Pegawai
*User* dapat mengedit data dari data master Jumlah Pegawai dengan cara mengklik tombol "Edit" pada kolom action seperti gambar dibawah ini:

[![tombol edit](../images/tombol-edit-data.png)](../images/tombol-edit-data.png)

Sehingga aplikasi akan menampilkan *Form Edit Jumlah Pegawai* seperti gambar dibawah ini:

[![Tampilan Edit Jumlah Pegawai](../images/form-kepegawaian-edit-pegawai.png)](../images/form-kepegawaian-edit-pegawai.png)
Gambar 23. Edit Jumlah Pegawai

Pada halaman ini *user* dapat melihat *form edit Jumlah Pegawai*, setelah mengedit data jumlah pegawai, *user* dapat mengklik "Submit" untuk menyimpan hasil editan kedalam database atau mengklik "Reset" untuk membersihkan data editan atau dapat mengklik tombol "Back" untu kembali ke halaman List Pegawai tanpa menyimpan data.

###### 2.2.3.1.3 Delete Data Jumlah Pegawai
Didalam List Pegawai *user* dapat menghapus jumlah pegawai yang tidak diinginkan dengan mengklik tombol "Delete" pada kolom action dari tabel list pegawai seperti gambar dibawah ini:
